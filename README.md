            <p><strong>Cartoon App</strong> is a fun, interactive mobile application built with Flutter, allowing users to explore a variety of cartoons, interact with characters, and enjoy an engaging visual experience. The app is available across multiple platforms, including Android, iOS, Web, and Desktop (Windows, macOS, Linux).</p>

                <li>Cross-platform support (Android, iOS, Web, Windows, macOS, Linux).</li>
                <li>User-friendly interface with intuitive navigation.</li>
                <li>Enjoy stunning visuals and animations.</li>

        <div class="section" id="installation">
            <h2>Installation</h2>
            <p>Follow these steps to get the project up and running on your local machine:</p>
            <ol>
                <li>Clone the repository:</li>
                <pre><code>git clone https://github.com/Tracy0708/cartoon_app.git</code></pre>
                <li>Navigate to the project directory:</li>
                <pre><code>cd cartoon_app</code></pre>
                <li>Install Flutter (if not already installed) by following the instructions from <a href="https://flutter.dev/docs/get-started/install" target="_blank">Flutter's official installation guide</a>.</li>
                <li>Install project dependencies:</li>
                <pre><code>flutter pub get</code></pre>
                <li>Run the app on your preferred platform:</li>
                <pre><code>flutter run</code></pre>
            </ol>
        </div>
